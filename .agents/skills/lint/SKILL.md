# lint 技能：知識圖譜健康巡檢流水線

## 操作流程
請參閱 `.claude/skills/lint/skill.md` 中的完整 SOP。兩者邏輯完全一致。

## 硬約束
- **僅讀掃描**：生成報告前，禁止修改、刪除、重命名任何檔案
- **手動確認**：報告後等待用戶確認再執行修復
- **靜默日誌**：修復完成後，在 wiki/log.md 追加操作紀錄
